# 🗺️ GhostSurf Roadmap

This document outlines the planned features, improvements, and goals for GhostSurf. It serves as a guide for contributors and users to track the project's development progress.

---

## ✅ Completed

- Project setup and initial repository structure
- Basic README, CONTRIBUTING, CONTRIBUTORS, ROADMAP, and CODE_OF_CONDUCT documentation
- Initial version of the core GhostSurf app

---

## 🚧 In Progress

- Refactor and clean up existing code
- Improve error handling and logging system

---

## 📝 Planned Features

- A service that will keep the internet traffic redirected over Tor after reboot
- Automatic Tor node display screen
- Custom Tor bridges usage

---

## 📢 Ideas & Suggestions

The following ideas are open for community discussion and proposals:

- GUI front-end (Qt)

Suggestions, feedback, and feature ideas are always welcome!
